Welcome to our Dynamic Grocery List Senior Project!

Project Goal 

            The goal of this project is to create a mobile application for Android based devices.
            The program will allow multiple users to access a centralized shopping list.
            The data on in the list will be dynamically updated based on individual usage.

Our Aim 

            This project is to simulate the development lifecycle of a program.
            As such, we will do our best adhear to the proceedures of development as closely as possible.

Targeted Demographic 

            The targeted group of individuals for this application would be from ages 10 and up.
            The application should be usable by anyone needing to make a list, but more so for groups of people living together.

Functionality

  • The user can download and install the program via the Google Play Store.

  • The user is able to create a new account if they do not have one.

  • If an account exists already, a user can log into the acount.

  • Multiple users will be able to access a singular application account.

  • The users will be able to create numerous lists.

  • Each list will be able to have string objects added to them.

  • When a list is being actively shopped, the application will notify the other users

  • The lists will be able to be edited by any user at any time.
